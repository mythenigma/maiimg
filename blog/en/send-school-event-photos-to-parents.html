Send School Event Photos to Parents (Easy Method)

Parents want simple access. Staff want less repetitive sending. One link solves both.

Interface snapshots

Typical issues

Too many resend requests
“Can you send again?” appears all day.
File overload
Chat groups become noisy quickly.
Different apps
Not every parent uses the same platform.
No closing rule
Old links remain active too long.

Practical setup

  1. Create one event set (up to 25 images per link).
  2. Share one direct link in parent group.
  3. Share QR on notice board or print sheet.
  4. Set expiry based on school timeline.
  5. Disable after the window closes.
