2) 必写功能清单（特别是 maiimg.html）

必须覆盖

2.1 上传与生成（maiimg.html）

  • 图片上传：支持 PNG/JPG/JPEG/GIF/WebP，单文件最大 50MB，最多 25 个文件。
  • 生成分享链接 + QR 码：上传后可生成 Gallery Link 与二维码。
  • 追踪码：生成访问记录 Code，并可跳转 tracking 页面查看访问记录。

2.2 访问控制（maiimg.html）

  • View limit（查看次数）+ per-view duration（每次查看时长）。
  • Expiration（到期时间）支持预设：1h / 3h / 24h / 5d / 自定义天数 / Unlimited。
  • 同时显示 UTC 到期时间和本地时区时间（用户可选时区）。

2.3 用户系统（maiimg.html）

  • 支持 Google 登录与邮箱登录/注册。
  • 邮箱验证、忘记密码、重发验证邮件。
  • 登录后可进入 dashboard.html 管理图集。

2.4 Dashboard 管理（dashboard.html）

  • 查看图集状态（Active / Deleted / Expired）、删除、恢复、追踪记录。
  • Replace 模式：选 A（目标）和 B（来源），复制 B 的内容/设置到 A，A 链接不变。
  • 手机版交互优化：A/B 大按钮、底部固定操作栏、顶部 A ← B 动态预览。

3) SEO 博文写作模板（推荐）

可直接套用
  1. 问题开场：用户在“图片分享/时效控制/访问追踪”上的痛点。
  2. 功能总览：MaiIMG 现在能做什么（上传、到期、登录、管理、追踪）。
  3. 3 步教程：上传图片 → 设置到期/访问规则 → 分享并追踪。
  4. 登录与不登录差异：匿名可用，登录后获得 Dashboard 管理能力。
  5. FAQ：到期后会怎样、如何恢复链接、如何看访问记录。
  6. CTA：去 maiimg.html 立即体验，或登录管理旧链接。

标题关键词建议

  • 主词：image hosting / temporary image link / QR image sharing / expiring image link
  • 长尾：how to share images with QR code and expiration / track image link access

文章里建议出现的功能词（按真实功能）

  • view limit
  • per-view duration
  • expiration presets
  • timezone-aware expiry
  • Google / email login
  • dashboard replace mode

5) 未登录用户提醒登录（Modal）规范

产品建议

目标：提醒匿名用户“登录可获得更好管理能力”，但不能频繁打扰。

  • 触发条件：用户未登录，且完成关键动作（例如成功生成链接）后触发。
  • 频控规则：1 小时内不重复提醒。
  • 关闭策略：用户点关闭后记录时间戳。
  • 登录后：不再弹。

推荐文案（英文基础）

  • Title: Get Better Control with an Account
  • Body: Sign in to manage galleries, restore links, and view access records from your dashboard.
  • Primary CTA: Sign In
  • Secondary CTA: Maybe Later

前端实现参考（localStorage 版）

const LOGIN_NUDGE_KEY = 'maiimg_login_nudge_last_ts';
const LOGIN_NUDGE_COOLDOWN_MS = 60 * 60 * 1000; // 1 hour

function shouldShowLoginNudge(isLoggedIn) {
  if (isLoggedIn) return false;
  const last = Number(localStorage.getItem(LOGIN_NUDGE_KEY) || 0);
  return Date.now() - last >= LOGIN_NUDGE_COOLDOWN_MS;
}

function markLoginNudgeShown() {
  localStorage.setItem(LOGIN_NUDGE_KEY, String(Date.now()));
}

// Example: after successful link generation
if (shouldShowLoginNudge(Boolean(sessionStorage.getItem('maiimg_jwt')))) {
  // show bootstrap modal
  markLoginNudgeShown();
}
