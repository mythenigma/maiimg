本次改了什么（Dashboard / Mobile Replace）

更新日期：2026-02-24

1) 手机版替换交互重做

已修改
  • 把卡片里的小圆点 A/B 改成大按钮：Set as A、Set as B（更容易点击）。
  • 选中后整卡高亮，并显示角标：Target A / Source B。
  • 同一条卡片不会同时作为 A 和 B（互斥逻辑保持）。

文件：dashboard.html

2) 新增手机底部固定操作栏

已修改
  • Replace 模式下，手机底部出现固定栏：步骤提示 + Cancel + Replace。
  • 步骤文案会动态变化： Step 1: Select target A → Step 2: Select source B → Ready to replace。
  • 只有 A 和 B 都选好，Replace 按钮才可点击。

文件：dashboard.html

3) 顶部 A ← B 替换预览增强（含动画）

已修改
  • 替换条改为更明确的流程显示：A #xxxx ← B #xxxx。
  • 箭头增加轻微脉冲动画，数字切换有 pop 动效。
  • 当 A/B 组合变化时，状态条会有短暂高亮动画，提醒用户关系已更新。

文件：dashboard.html

4) 英文基础文案 + i18n 键补充（全语言）

已修改
  • 新增文案统一以英文为默认基础。
  • 已为所有支持语言补齐移动替换相关键，包含： mobileRoleTarget、mobileRoleSource、mobileSetTarget、mobileSetSource、mobileStep1、mobileStep2、mobileHint、mobilePickedTarget、mobilePickedSource、mobileReady、mobileReadySummary。

文件：i18n.js

现状
