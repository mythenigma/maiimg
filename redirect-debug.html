问题： Cloudflare Pages 重写 URL 时丢失了原始路径信息

解决： 修改 _redirects 规则，将 ID 作为查询参数传递

🔄 新的重定向规则

之前： /pic/* /pic-redirect.html 200

现在： /pic/:id /pic-redirect.html?id=:id 200

效果： /pic/CF → /pic-redirect.html?id=CF
